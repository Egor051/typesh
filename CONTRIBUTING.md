# Contributing

Спасибо за интерес к проекту.

## Локальный запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python main.py
```

## Рекомендации по изменениям

- держите изменения небольшими и изолированными;
- не коммитьте секреты (`DISCORD_TOKEN`, реальные `.env`);
- для изменений парсера проверяйте сценарии на `debug_*.html`;
- перед PR убедитесь, что проект запускается и импортируется без ошибок.

## Checklist перед PR

- [ ] обновлены docs при изменении поведения;
- [ ] не изменена публичная конфигурация без отражения в `.env.example` и `README.md`;
- [ ] локальный запуск проходит без ошибок.
